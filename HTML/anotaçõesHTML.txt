AULA 02:

META TAGS PADRÃO:
    - meta charset  = definir o tipo de caracteres do site ( o padrão é o UTF-8 ) quando se usa o UTF-8 não vai ter problema com acentos
    - meta http-equiv = compatibilidade com a internet explorer ( o padrão é X-UA-Compatible ) e ( IE=edge ) so server pra compatibilidade
    - meta nome = ( o padrão é viewport ) que seria a area visivel do site, que ela vai se adaptar com telas diferentes ( mobile, pc, etc... )

METAS TAGS " EXTERNAS ":
    - <meta nome="description" content="Esse é o meu primeiro teste"> ; ( descrição do site quando o link é enviado )
    - <meta name="keywords" content="testes, front-end, testando, primeiro site, HTML"> ; ( tags/palavras chaves do site )
    - <meta name="author" content="Podin | wesley"> ; ( autor do site, ava '_' )

TAGS DE CABEÇALHO:
    - <h1> teste1 </h1> ( maior titulo de CABEÇALHO )
    - <h2> teste2 </h2> ( um titulo um pouco menor doq o h1 )
    - <h3> teste3 </h3> ( um titulo um pouco menor doq o h2 )
    - <h4> teste4 </h4> ( um titulo um pouco menor doq o h3 )
    - <h5> teste5 </h5> ( um titulo um pouco menor doq o h4 )
    - <h6> teste6 </h6> ( um titulo um pouco menor doq o h5 )

TAGS DE PARÁGRAFO:
    - <p> pa1 </p>
    - p>Lorem ( enter ) ; ( um texto em Latim )
    - p*x ( X tags <p> de uma vez só )

AULA 03:

TAGS DE FORMATAÇÃO DE TEXTO:
    - <b> Texto em negrito </b> ( colocar palavras em negrito )
    - <strong> dá ENFASE a tal frase </strong> ( é so pra maquina interpretar melhor )
    - <hr>  </hr> ( literalmente uma linha )
    - <br>  </br> ( quebra linha )
    - <i>  </i> ( deixa a palavra DEITADA )
    - <u>  </u> ( deixa sublinhado )
    - <small>  </small> ( deixa a palavra pequena )
    - <del>  </del> ( deixa um " risco " no meio da palavra )
    - <strike>  </strike> ( só coloquei pra ficar completo, pois os navegadores não usam mais o strike '-' )
    - <sup>  </sup> ( deixa o número pequeno, tipo expoensisasão )
    - <sub>  </sub> ( é a msm coisa do SUB mas o número fica em baixo )
    - <mark>  </mark> ( literalmente um marca texto )
